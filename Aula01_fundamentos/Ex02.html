HTML

A linguagem HTML, Hypertext Markup Language é uma linguagem de marcação de conteúdo. Tem por objetivo definir a estrutura do conteúdo, como por exemplo: definir títulos, paragrafos, listas, tabelas, imagens, audio, video, liks, entre outros.
