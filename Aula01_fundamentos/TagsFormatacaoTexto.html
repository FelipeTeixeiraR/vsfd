Explicação de diferentes Tags para a formatação de texto:

Del: indica um texto que doi removido de um documento, ficando riscado:

Este é um texto onde a tag del foi aplicada.

Ins: indica um texto que foi inserido no documento, ficando sublinhado:

Este é um texto onde a tag ins foi aplicada.

sub: exibe o texto como subescrito:

Este é um texto onde a tag sub foi aplicada.

sup: exibe o texto como sobreescrito:

Este é um texto onde a tag sup foi aplicada.

blockquote: destaca uma citação longa de outra fonte:

Este é um texto onde a tag blockquote foi aplicada.

q: marca uma citação curta, exibida entre aspas:

Este é um texto onde a tag q foi aplicada.

abbr: define uma abreviação ou sigla, mostrando o significado ao passar o mouse:

Passe o mouse por cima deste texto

address: indica informações de contato, como o do autor da pagina:

Rua fulanlda da silva almeida da costa, numero 1234

.

cite: usado para citar o título de uma obra:

A viagem ao funda da terra.

bdo: define a direção do texto de direita para esqueda ou de esquerda para direita:

Texto escrito para esquerda
Texto escrito para a direita

code: exibe como se fosse um codigo de computador:

function exemplo() {
console.log("Olá, mundo!");
}

pre: Exibe texto pré-formatado, mantendo espaços e quebras de linha:

a quebra de linha
    foi
    aplicada sem o uso do
    br.

kbd: Indica entrada de teclado:

Pressione a letra B (nada acontece só indica o que deve ser precionado).
